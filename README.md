# BigQuery Public Goods MCP Hub
> Modern tooling for community public goods, powered by Google BigQuery.

Welcome to the **BigQuery Public Goods Hub**. This repository is a collection of Model Context Protocol (MCP) servers designed to provide high-fidelity, cost-efficient access to blockchain settlement data via Google BigQuery.

## 📦 Available Packages

### [Polymarket PubGoods MCP](./packages/bigquery-polymarket-pubgoods-mcp)
A comprehensive discovery and audit engine for Polymarket on-chain settlement data. Includes 14 tools for tracking market pulse, counterparty movements, and capital flows.

## 📂 Repository Structure

- `packages/`: Independent MCP server implementations.
- `docs/`: Technical architecture, AI agent skill playbooks, and integration guides.

## 🛡️ Public Goods Mission
This project leverages the **Google Web3 Public Dataset Program** to provide developers and researchers with direct, high-fidelity access to on-chain settlement data.
